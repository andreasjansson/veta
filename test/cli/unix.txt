%platform unix

=====
veta works from subdirectory
=====
veta init --reinitialize
veta add --title "Test note" --tags "test" --body "Test body"
mkdir -p subdir/nested && cd subdir/nested && veta ls
-----
Reinitialized veta database in {{ path }}
Added note 1
1: Test note ({{ modified }}) -- Test body
-----
where
* path matches /\.veta[\/\\]db\.sqlite/
